# AI-ML Assignments

A collection of internship and practice assignments created using **Google Colab** to explore machine learning and data science concepts.

## About

This repository contains beginner-friendly AI and ML assignments completed during internship practice sessions. Each task is designed to build practical skills through hands-on experimentation and real-world problem solving.

## Topics Covered

- Data preprocessing and cleaning
- Classification and regression tasks
- Natural language processing (NLP)
- Image processing and computer vision basics
- Exploratory data analysis (EDA)
- Model evaluation and interpretation

## Tools & Technologies

- Google Colab
- Python
- NumPy
- pandas
- scikit-learn
- Jupyter notebooks

## Repository Structure

- `*.ipynb.url` files: Colab assignment links
- `README.md`: project overview and repository guide

## Learning Outcomes

- Practical experience with AI/ML workflows
- Better understanding of common ML algorithms
- Improved data cleaning and visualization skills
- Hands-on practice with Python-based machine learning tools
- Confidence to explore more advanced AI projects
